# KJV Rightly Dividing — Quick Start (5 minutes)

Get from zero to studying in five steps.

## 1. Clone or download the project

```bash
git clone <your-repo-url> kjv-rd && cd kjv-rd
```

Or unzip the project folder and open a terminal in it.

## 2. Install dependencies

```bash
npm install
```

## 3. Run the app

```bash
npm run dev
```

## 4. Open in browser

Go to **http://localhost:3000**

## 5. Ask a question

- Type in the text box, e.g. **"How are we saved today?"** or **"Should Christians tithe?"**
- Click **Select verses**
- Read the verses grouped by category (Paul's Grace, Jesus' Kingdom Offer, Law/Israel, etc.)

---

**That’s it.** All verses are KJV. For more detail, see [Tutorial](TUTORIAL.md) and [Manual](MANUAL.md).
